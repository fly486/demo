# Beidou Simple Example
=================

### Install

```
$ yarn install
```

### Start  

``` 
$ npm start
```  

then go to http://127.0.0.1:6001/

### Basic Usage

* add a new page in /client folder, eg. `/client/test.jsx`
* start app: `npm start`
* go to http://127.0.0.1:6001/test
